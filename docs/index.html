Questions, answered

If the source is open, why pay?

Because compiling Python with PyInstaller + dealing with Windows code-signing + supporting the binary takes time. $9.99 saves you 10 minutes and supports indie development. If you'd rather build it yourself, the instructions are in the README — totally fine.

What about the SmartScreen warning on first run?

Windows shows "Windows protected your PC" for any unsigned exe — this is a generic OS warning, not a virus alert. Click "More info" → "Run anyway." Code signing certificates cost ~$200/year; we'll add one once revenue justifies it.

Does it work on Windows 11?

Yes. Tested on Windows 10 and Windows 11. Requires admin rights (UAC prompt on launch).

Will it touch my personal files?

No. The list of paths is hardcoded in core/tasks.py — you can read it on GitHub. Documents, Desktop, Pictures, and program installs are never touched. Nothing is deleted until you click Confirm.
